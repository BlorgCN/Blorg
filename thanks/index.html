程序：Gridea

Gridea 是一个静态博客写作客户端，可以帮助博主更容易地构建并管理博客或任何静态站点。主要有如下特性👇

📝 你可以使用最酷的 Markdown 语法，进行快速创作

🌉 你可以给文章配上精美的封面图和在文章任意位置插入图片

🏷️ 你可以对文章进行标签分组

💻 你可以在 Windows，MacOS 或 Linux 设备上使用此客户端

🌎 你可以使用 𝖦𝗂𝗍𝗁𝗎𝖻 𝖯𝖺𝗀𝖾𝗌 或 Coding Pages 向世界展示，未来将支持更多平台

💬 你可以进行简单的配置，接入 Gitalk 或 DisqusJS 评论系统

🇬🇧 你可以使用中文简体或英语

🌁 你可以任意使用应用内默认主题或任意第三方主题，强大的主题自定义能力

🖥 你可以自定义源文件夹，利用 OneDrive、百度网盘、iCloud、Dropbox 等进行多设备同步

博客联盟·新版的搭建采用了Gridea作为程序，以此实现整站静态化并且部署到阿里云CDN，极大地加快了访问速度。

字体：邹乐伟

博客联盟·新版的UI设计由于才用了成熟稳重的设计风格，因此在设计过程中我们尝试了无数种字体，最终对比之下，最适合这种稳重设计风格的字体只有宋体这一类的字体。

考虑到用户的访问设备不一定安装有相应字体，为了保障每一位用户访问的体验相同，我们决定在网站嵌入字体文件。但事实上大部分的宋体如仿宋、华宋等等，字体文件都比较大，做字体嵌入不太合适。

在寻找了无数个宋体品种之后，我们发现了这个字体：

欣意吉祥宋

这款字体不仅评论，而且体积较小，符合我们的选择要求。同时他是一款支持免费商用的字体哦~

欣意吉祥宋是由左字字库出品的一套宋体字体，作者邹乐伟。整套字形视觉感偏瘦高，宽高比例为4:5，尝试过1:1甚至更矮的比例，不够优雅，因为像这种圆润纤细古风女性化的字体，都是比较修长才好看，最后才决定4:5这个近乎完美的比例。

这款字风格圆润，比如横画开头及结尾由传统字体的三角尖凸变成较圆润的风格，竖画由三角尖凸也变成圆润无尖凸风格，竖勾撇捺点的笔画同理。
